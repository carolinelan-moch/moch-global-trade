# MOCH GLOBAL TRADING LIMITED NZ Homepage

Static GitHub Pages homepage project.

## Structure

- `index.html`
- `style.css`
- `app.js`
- `images/`

## How to use

1. Upload all files to your GitHub repository.
2. Enable GitHub Pages in repository settings.
3. Replace placeholder social media links in `index.html` footer.
4. Replace `contact@mochglobaltrading.co.nz` if you use a different email.

## Notes

The contact form currently uses a static `mailto:` solution suitable for GitHub Pages. It can be upgraded later to Formspree, Netlify Forms, or a custom backend API.
